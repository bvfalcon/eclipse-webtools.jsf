Adding and Removing JSF Library References from a Project


JSF Library References for a project are managed as properties of the project. To add or remove library referernces: Project -> Properties -> JSF Library References

  • To modify the choice of implementation, choose from the Implementation dropdown list.
  • To add a component library, select the library from list on left and use ">" to add to the selected list
  • To add all libraries, use ">>"
  • To remove a component library, select library from list on the right and use "<" button
  • To remove all libraries, use "<<" button
  • To create a new library, use "New...: button
  • To change whether or not a selected component library is deployed, use the "Deploy" checkbox

Click "OK" button to complete the update of the JSF Library References for this project
