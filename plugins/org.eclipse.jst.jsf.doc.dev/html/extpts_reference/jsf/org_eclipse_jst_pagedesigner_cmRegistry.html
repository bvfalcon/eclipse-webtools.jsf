Since:
2.0

Description:

This extension point is provisional and subject to change

Configuration Markup:

<!ELEMENT extension (registry*)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT registry EMPTY>

<!ATTLIST registry

configFile CDATA #IMPLIED

class      CDATA #IMPLIED

uri        CDATA #REQUIRED>

User must provide either "configFile" attribute or "class" attribute.


  • configFile - A path relative to the root of the declaring plugin. Point to a config file that contains the information for the specified tag lib of the uri.
  • class - A class that implements ICMRegistry. The implemented ICMRegistry must return the same uri as the uri attribute in getSupportedURI() method.
