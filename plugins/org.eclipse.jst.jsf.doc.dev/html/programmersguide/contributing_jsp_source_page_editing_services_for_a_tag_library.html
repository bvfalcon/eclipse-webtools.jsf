Contributing JSP Source Page Editing Services for a Tag Library

The JSF Tooling Team has provided several frameworks for improving the user experience of developing JSP pages using the WTP SSE editor. The net result is that:

  • Content Assist
  • Validation

for the attribute values of a tag library have been provided and implemented for the JavaServer Faces standard tag libraries.

This has been accomplished by providing frameworks for supplying additional meta-data for tag libraries that can be used with classes for using the meta-data for providing source editing services.

Future releases of WTP, by using these frameworks, should allow for additional meta-data provided services like:

  • Quick Fix/Quick Assist
  • Hyperlink to Referenced Object Editor
  • Others? Suggestions?

It is hoped that adopters will use these frameworks to provide improved source editing user experiences using tag libraries other than the JSF standard tag libraries.
