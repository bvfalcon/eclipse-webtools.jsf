FAQ

  1. How to make the Visual Page Designer the default editor for JSP pages?
    2.
    The JSF Tools Project-contributed Visual Page Designer (VPD) is NOT registered as the default editor for JSP files. You can make the VPD the default editor for JSP files by following these steps:
    • Navigate to "Window" -> "Preferences..." -> "General" -> "Editors" -> "File Associations"
    • In the "File types" section, select "*.jsp"
    • In the "Associated editors" section, select "Web Page Editor"
    • Select the "Default" button
